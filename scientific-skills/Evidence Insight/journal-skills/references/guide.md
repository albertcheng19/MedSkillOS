# Journal Recommendation Reference List

## Input List
- Paper Title and Abstract
- Subject and Keywords
- Submission Preferences (Open Access, Review Cycle)

## Output List
- Journal Recommendation List
- Matching Rationale and Precautions
- Alternative Journal List

## Output Format
- Table Columns: Journal / Matching Rationale / Impact Factor or Quartile (Optional) / Review Cycle / Open Access Risk / Priority
- Note whether further verification of submission requirements is needed

## Quality Check
- Recommendation rationale is consistent with the research topic
- Avoid non-compliant or predatory journals

## Common Issues
- Focusing only on metrics while ignoring journal scope
- Failure to check submission requirements
