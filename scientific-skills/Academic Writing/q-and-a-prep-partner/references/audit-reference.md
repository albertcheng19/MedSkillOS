# Audit Reference

## Scope

- Presentation Q&A preparation
- Anticipated-question drafting
- Response framing and rehearsal support

## Supported Audit Paths

- `python -m py_compile scripts/main.py`
- `python scripts/main.py --help`

## Fallback Boundary

If talk or abstract inputs are incomplete, the skill should still return:

- the missing presentation context
- the questions that can still be drafted safely
- assumptions that need confirmation before rehearsal
- the next checks before accepting the final Q&A set
