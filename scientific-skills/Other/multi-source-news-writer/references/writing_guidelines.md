# Writing Guidelines

## 1. Information Integration
- **Fact Extraction**: Extract key facts (5W1H elements) from all sources.
- **De-duplication**: Identify and remove repetitive information.
- **Consolidation**: Merge similar viewpoints, retaining the most authoritative expression.
- **Conflict Handling**: Mark contradictory information and note points needing verification.

## 2. Structural Optimization (Inverted Pyramid)
- **Headline**: Extract the most newsworthy core fact (max 20 characters).
- **Lead**: Include the core event + latest progress (approx. 100 characters).
- **Body**: Sort by importance descending, including:
    - Event Background (max 100 characters)
    - The Event Itself (balanced presentation of multiple perspectives)
    - Future Impact/Trends
- **Total Length**: Approx. 1000 characters for the body.

## 3. Style Specifications
- **Tone**: Objective and neutral.
- **Style**: AP Style (short sentences, active voice).
- **Data**: Use unified quantification units.
- **Quotes**: Retain key figures' original words with identity attribution. If no key figures, omit this section.

## 4. Formatting Output
Use the following Markdown structure:

```markdown
## [Headline]

### [Lead Paragraph]

[Body Paragraphs with subheadings]
```
