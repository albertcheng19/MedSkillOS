# Literature Extensive Reading Reference List

## Input List

- Full text of the literature (Markdown, usually containing `## Page XX` pagination)
- Research topic or focus (optional)

## Output List

- Structured brief summary (research question/method/results/conclusion)
- Judgment and reasons for whether it is worth intensive reading
- Output files are located in the `outputs/` folder under this skill directory

## Output Format

- Use sections or tables to output structured content.
- A concise summary within 5-8 sentences per literature.
- Can include 1-2 sentences of intensive reading suggestions.
- Use "<Literature Name> Literature Extensive Reading" as the title; prioritize the input filename for the literature name, followed by the literature title.
- Output as a UTF-8 encoded Markdown document.
- Save output files to the `outputs/` folder under this skill directory (e.g., `literature-extensive-read/outputs/`).

## Quality Check

- Research questions, methods, results, and conclusions are complete.
- Do not exaggerate or add content not mentioned in the text.
- Language is concise and easy to read quickly.

## Common Issues

- Only listing background without mentioning methods and results.
- Ignoring research limitations or scope of application.
