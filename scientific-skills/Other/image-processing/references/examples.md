# Examples

## Example 1: Convert to WebP

```
python scripts/convert_images.py --source-dir "D:\Images" --output-dir "D:\ImagesWebp" --format webp --quality 80
```

## Example 2: Convert to JPG and overwrite

```
python scripts/convert_images.py --source-dir "D:\Images" --output-dir "D:\ImagesJpg" --format jpg --quality 85 --overwrite
```

## Example 3: Recursive conversion

```
python scripts/convert_images.py --source-dir "D:\Images" --output-dir "D:\ImagesOut" --recursive
```
