# ArXiv API Reference

## Official Documentation
- **API User Manual**: [https://arxiv.org/help/api/user-manual](https://arxiv.org/help/api/user-manual)
- **Python Library**: [https://github.com/lukasschwab/arxiv.py](https://github.com/lukasschwab/arxiv.py)

## Query Syntax
- **Prefixes**:
    - `ti`: Title
    - `au`: Author
    - `abs`: Abstract
    - `co`: Comment
    - `jr`: Journal Reference
    - `cat`: Category (e.g., `cs.AI`, `physics:hep-th`)
    - `rn`: Report Number
    - `id`: Id (e.g., `cond-mat/0709123`)
    - `all`: All of the above

- **Boolean Operators**:
    - `AND`
    - `OR`
    - `ANDNOT`

## Categories
Common Computer Science categories:
- `cs.AI`: Artificial Intelligence
- `cs.CL`: Computation and Language
- `cs.CV`: Computer Vision
- `cs.LG`: Machine Learning
- `cs.SE`: Software Engineering
