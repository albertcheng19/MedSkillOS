# Cell Press Writing Style Guide

This guide serves as a comprehensive writing reference for Cell, Neuron, Immunity, Molecular Cell, Developmental Cell, Cell Reports, and other journals under the Cell Press umbrella.

**Last Updated**: 2024

---

## Overview

Cell Press journals emphasize **mechanistic depth**, **rigorous experimentation**, and **biological insight**. Unlike Nature/Science, which prioritize broad readability, Cell series papers are written for biologists who value technical details and comprehensive data.

### Core Philosophy

> "Cell papers tell a complete mechanistic story supported by exhaustive experimentation."

**Main Goal**: To provide deep biological insights and advance the understanding of fundamental mechanisms through extensive experimental validation.

---

## What Makes Cell Press Unique

Cell Press includes several unique elements not commonly found in other journals:

### 1. Summary (Instead of Abstract)

Cell uses "Summary" instead of "Abstract"—while functionally similar, it places more emphasis on synthesis.

### 2. Graphical Abstract (Mandatory)

A visual summary that appears in the table of contents. **This is a mandatory requirement for all Cell Press journals.**

### 3. eTOC Blurb

A 30-50 word "elevator pitch" style introduction prepared for the electronic Table of Contents.

### 4. Highlights

3-4 bullet points (≤85 characters each, including spaces) used to summarize core findings.

### 5. In Brief

A single sentence summarizing the entire paper.

---

## Audience and Tone

### Target Audience

- Expert-level biologists in related fields
- Readers familiar with relevant techniques and terminology
- Expectation of comprehensive data and mechanistic depth
- Value placed on rigor and reproducibility

### Tone Characteristics

| Feature | Description |
|---------------|-------------|
| **Technical** | Uses appropriate professional terminology for the field |
| **Mechanistic** | Focuses on "how" and "why," not just "what" |
| **Comprehensive** | Explores problems deeply and thoroughly |
| **Data-rich** | Provides extensive experimental support |
| **Precise** | Uses accurate terminology and quantitative data |

### Voice

- **First person ("we") is acceptable**: "We demonstrate that..."
- **Active voice is encouraged**: "We identified..."
- **Confident but measured**: Strong claims must be supported by strong evidence

## Summary

### Style Requirements

- Cell requires a **maximum of 150 words**; other Cell Press journals vary
- **Fluid paragraph** (non-structured sections)
- **Information-dense**: Every sentence should convey a key point
- **Mechanistic focus**: What was discovered and how it works

### Summary Structure

1. **Context** (1 sentence): The biological problem/puzzle
2. **Approach** (1 sentence): What you did
3. **Key findings** (2-4 sentences): Main results with mechanisms
4. **Significance** (1 sentence): What biological principle this reveals

### Summary Example (Cell Style)

```
Cellular senescence is a stress response that arrests proliferation and 
promotes tissue remodeling, but the mechanisms controlling senescent cell 
fate remain unclear. Here, we identify the transcription factor FOXO4 as a 
critical regulator of senescent cell viability. FOXO4 is highly expressed 
in senescent cells and sequesters p53 away from mitochondria, preventing 
apoptosis. Using a cell-penetrating peptide that disrupts FOXO4-p53 
interaction, we selectively induce senescent cell apoptosis in vitro and 
in vivo. Administration of this peptide to aged mice restores fitness, fur 
density, and renal function. These findings reveal FOXO4-p53 as a senescence 
vulnerability and establish proof-of-concept for targeted senolytic 
interventions in aging.
```

## Graphical Abstract

### Purpose

To provide a visual summary in a single-figure format for the table of contents, conveying the paper's complete message.

### Requirements

- **Dimensions**: Square format, typically 1200 × 1200 pixels
- **Layout**: Clean and uncluttered
- **Content**: Shows workflow, key findings, and mechanism
- **Text**: Minimal labels, large and legible font
- **Color**: Vibrant and professional

### Design Elements

```
Typical Graphical Abstract components:
1. Starting point (cell, organism, condition)
2. Intervention/Treatment (arrows, symbols)
3. Key measurements or observations
4. Results/Conclusions (visual representation)
5. Minimal text labels connecting elements
```

### Example Description (for sketch generation)

```
"Graphical abstract showing: Left panel - normal cells with FOXO4 (blue) 
and p53 (green) separate. Center panel - senescent cells with FOXO4 
binding p53, preventing apoptosis. Right panel - FOXO4 peptide disrupts 
interaction, allowing p53 to reach mitochondria, triggering apoptosis. 
Arrow at bottom showing aged mouse → treatment → rejuvenated mouse."
```

## Highlights

### Format

3-4 bullet points, each ≤85 characters (including spaces)

### Content Guidelines

- Start with a verb or key noun
- Include specific findings
- Ensure each highlight can stand alone
- Cover different aspects of the paper

### Highlights Example

```
• FOXO4 is selectively expressed in senescent cells

• FOXO4 sequesters p53, preventing senescent cell apoptosis

• A FOXO4-targeting peptide induces selective senescent cell death

• Senolytic peptide treatment restores function in aged mice
```

## eTOC Blurb

### Format

30-50 words, used for the electronic table of contents

### Writing Guidelines

- Written by the author (editors may modify)
- Start with the author's name or the key finding
- Form a complete and engaging sentence
- Highlight the most exciting part

### eTOC Blurb Example

```
Baar et al. identify FOXO4 as a vulnerability of senescent cells and 
develop a peptide that induces targeted apoptosis of senescent cells. 
Treatment of aged mice with this senolytic peptide restores fitness 
and organ function.
```

## Introduction

### Length and Structure

- **4-6 paragraphs** (800-1200 words)
- More comprehensive than Nature/Science
- Can include more technical details and literature citations

### Paragraph-by-Paragraph Guide

**Para 1: Biological Context**
- Establish the biological process or system
- Why is understanding this important?
- Introduce key players and mechanisms

**Para 2-3: State of the Field**
- Detailed review of relevant prior work
- Establish known mechanisms
- More synthetic than Nature/Science

**Para 4: Knowledge Gap**
- What is still unknown or controversial?
- Why is this a critical question?
- What has hindered progress?

**Para 5: Your Approach**
- How did you address this problem?
- What techniques/systems did you use?
- Why is your approach appropriate?

**Final Para: Preview of Key Findings**
- Briefly state what you found
- How does this move the field forward?
- Set the stage for the structure of the Results section

### Introduction Paragraph Example

```
Cellular senescence is characterized by stable cell-cycle arrest, profound 
chromatin alterations, and a complex secretory phenotype known as the 
senescence-associated secretory phenotype (SASP) (Coppé et al., 2008; 
Rodier and Campisi, 2011). Senescent cells accumulate with age and at 
sites of pathology, where they can drive tissue dysfunction through 
SASP-mediated inflammation and disruption of tissue architecture (van 
Deursen, 2014). The targeted elimination of senescent cells—senolysis—has 
emerged as a promising therapeutic strategy, with genetic and pharmacological 
approaches demonstrating benefits in mouse models of aging and age-related 
disease (Baker et al., 2011, 2016; Chang et al., 2016).
```

## Results

### Organization

Cell papers typically have **5-8 Results subsections**, each with a descriptive subheading:

```
Results
├── Section 1: Discovery of the phenomenon
├── Section 2: Characterization of the mechanism  
├── Section 3: Identification of molecular players
├── Section 4: Functional validation
├── Section 5: In vivo confirmation
├── Section 6: Therapeutic proof-of-concept
└── Section 7: Broader implications
```

### Subheading Style

Cell uses **declarative subheadings** to state findings:

❌ "Analysis of FOXO4 expression" (Descriptive - avoid)
✅ "FOXO4 Is Selectively Upregulated in Senescent Cells" (Declarative)

### Results Writing Style

- **Exhaustive detail**: Cell expects more experimental context in Results than Nature
- **Figure-based narrative**: Each main figure usually corresponds to one Results subsection
- **Statistical rigor**: All quantitative data must be accompanied by statistical analysis
- **Biological interpretation**: Cell incorporates more interpretive text compared to pure result descriptions

### Results Paragraph Example

```
To identify transcription factors regulating senescent cell viability, we 
performed RNA sequencing on proliferating and senescent human fibroblasts 
(IMR90 cells induced to senesce by replicative exhaustion, ionizing 
radiation, or oncogene-induced senescence). Differential expression 
analysis revealed 47 transcription factors significantly upregulated 
across all senescence modalities (FDR < 0.05, fold change > 2; Figure 1A 
and Table S1). Among these, FOXO4 showed the highest and most consistent 
upregulation (12.3 ± 2.1-fold; Figure 1B), a finding we confirmed by 
quantitative RT-PCR (Figure 1C) and immunoblot analysis (Figure 1D). 
Immunofluorescence microscopy revealed nuclear FOXO4 accumulation in 
senescent but not proliferating cells (Figure 1E,F).
```

## Discussion

### Structure

The Discussion section in Cell is **thorough and mechanistic**:

**Para 1: Summary**
- Restate key findings
- Synthesize the main message

**Para 2-4: Mechanistic Interpretation**
- Deep dive into how your findings fit into known biological logic
- Propose a model
- Discuss molecular mechanisms in detail

**Para 5: Comparison with Literature**
- How do your findings relate to prior work?
- Resolve apparent contradictions

**Para 6: Implications and Applications**
- Therapeutic significance
- Broader scientific value

**Para 7: Limitations**
- Honest assessment
- Unresolved questions

**Final Para: Conclusion**
- Big-picture take-home message
- Future directions

## Experimental Procedures / STAR Methods

### STAR Methods Format

Cell uses a structured **STAR Methods** section:

```
RESOURCE AVAILABILITY
  Lead Contact
  Materials Availability
  Data and Code Availability

EXPERIMENTAL MODEL AND SUBJECT DETAILS
  Cell Lines
  Animals
  Human Subjects

METHOD DETAILS
  [Detailed protocols for each technique]

QUANTIFICATION AND STATISTICAL ANALYSIS
```

Cell requires a detailed table of all critical resources:

## Figures

### Figure Philosophy

Cell papers are typically **figure-intensive**, containing large multi-panel figures:

- Usually contains **6-8 main figures**
- **Multi-panel format**: It is common for a single figure to contain 6-12 panels
- **High data density**: Comprehensive experimental support
- **Extended Data**: Supplemental figures used for additional validation

### Panel Labeling

Panels are labeled with lowercase letters: **(A)**, **(B)**, **(C)**

### Figure Legend Format

```
Figure 3. FOXO4 Sequesters p53 in the Nucleus of Senescent Cells

(A) Immunofluorescence microscopy of p53 (green) and FOXO4 (red) in 
proliferating (left) and senescent (right) IMR90 cells. DAPI (blue) 
marks nuclei. Scale bar, 10 μm.

(B) Quantification of nuclear p53 intensity in proliferating versus 
senescent cells. Data represent mean ± SEM; n = 3 biological replicates, 
>100 cells per condition. ***p < 0.001, two-tailed Student's t test.

(C and D) Co-immunoprecipitation of FOXO4 and p53 in proliferating (C) 
and senescent (D) cell lysates. IgG, immunoglobulin G control.

(E) Proximity ligation assay for FOXO4-p53 interaction. Red dots indicate 
interaction events. Scale bar, 10 μm.

(F) Model of FOXO4-mediated p53 sequestration in senescent cells.

See also Figure S3 and Table S2.
```

## References

### Citation Style

- **Author-year format**: (Smith et al., 2023) or Smith et al. (2023)
- **Multiple citations**: (Smith et al., 2020; Jones et al., 2021)
- **Two authors**: (Smith and Jones, 2023)
- **Three or more authors**: (Smith et al., 2023)

### Reference List Format

```
Baker, D.J., Wijshake, T., Tchkonia, T., LeBrasseur, N.K., Childs, B.G., 
van de Sluis, B., Kirkland, J.L., and van Deursen, J.M. (2011). Clearance 
of p16Ink4a-positive senescent cells delays ageing-associated disorders. 
Nature 479, 232–236.
```

## Common Mistakes

1. **Insufficient mechanism**: Describing what happened without explaining how.
2. **Inadequate experimental controls**: Missing critical control groups.
3. **Weak phenotype validation**: Using only a single method instead of multiple approaches for validation.
4. **Lack of in vivo work**: Cell papers are generally expected to include animal studies.
5. **Incomplete figure panels**: Failing to show all relevant experimental conditions.
6. **Missing graphical abstract**: This is a mandatory requirement.
7. **Highlight character limit exceeded**: Each point must be ≤85 characters.

## Pre-submission Checklist

### Essential Elements
- [ ] Graphical abstract (square format)
- [ ] Highlights (3-4 points, ≤85 characters each)
- [ ] eTOC blurb (30-50 words)
- [ ] Summary (≤150 words)
- [ ] Key Resources Table

### Content
- [ ] Maintain mechanistic depth throughout
- [ ] Employ multiple complementary approaches
- [ ] In vivo validation (if applicable)
- [ ] Use declarative subheadings
- [ ] Detailed figure panels

### Style
- [ ] Technical precision in terminology
- [ ] Use author-year citation style
- [ ] Complete and standalone figure legends
- [ ] Correct STAR Methods format
