# ISO 13485:2016 Gap Analysis Checklist

This comprehensive checklist is designed to help you identify gaps between your current Quality Management System (QMS) and the requirements of the ISO 13485:2016 standard.

## How to Use This Checklist

**Status Indicators:**
- ✅ **Compliant:** Requirement is fully implemented and documented
- ⚠️ **Partial:** Requirement is partially implemented; improvements needed
- ❌ **Non-compliant:** Requirement is not implemented or not documented
- N/A **Not Applicable:** Requirement does not apply (justification must be provided)

**For each item:**
1. Assess the current status
2. Identify existing documentation/evidence
3. Record gaps or deficiencies
4. Prioritize required actions
5. Assign a responsible person and set a target date
