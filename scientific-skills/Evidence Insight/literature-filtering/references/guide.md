# Literature Filtering Reference Checklist

## Input Checklist

- Original literature list
- Year range and journal list
- Document types and language restrictions

## Output Checklist

- List of included literature
- List of excluded literature with reasons
- Screening statistics summary

## Output Format

- Table listing literature and screening results.
- Consistency in the "Reason" field.

## Quality Check

- Screening rules are applied consistently across all literature.
- Clear reasons for handling borderline cases.

## Common Issues

- Missing exclusion reasons make re-evaluation difficult.
- Inconsistency in rules.
