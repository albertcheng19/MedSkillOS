# QUADAS-2 Assessment Criteria

## Domain 1: Patient Selection

**Risk of Bias Signaling Questions:**
1.  **Was a consecutive or random sample of patients enrolled?**
    *   Yes: Explicitly stated consecutive or random.
    *   No: Non-consecutive or convenience sampling.
    *   Unclear: Not reported.
2.  **Was a case-control design avoided?**
    *   Yes: Cohort or cross-sectional design.
    *   No: Case-control design (known cases vs controls).
3.  **Did the study avoid inappropriate exclusions?**
    *   Yes: All eligible patients included; "difficult to diagnose" patients not excluded.
    *   No: Exclusions that could bias results.

## Domain 2: Index Test

**Risk of Bias Signaling Questions:**
1.  **Were the index test results interpreted without knowledge of the results of the reference standard?**
    *   Yes: Blinded interpretation.
    *   No: Unblinded.
2.  **If a threshold was used, was it pre-specified?**
    *   Yes: Threshold defined before analysis.
    *   No: Optimal threshold selected after analysis.

## Domain 3: Reference Standard

**Risk of Bias Signaling Questions:**
1.  **Is the reference standard likely to correctly classify the target condition?**
    *   Yes: Gold standard or best available method.
    *   No: Flawed reference standard.
2.  **Were the reference standard results interpreted without knowledge of the results of the index test?**
    *   Yes: Blinded interpretation.
    *   No: Unblinded.

## Domain 4: Flow and Timing

**Risk of Bias Signaling Questions:**
1.  **Was there an appropriate interval between index test and reference standard?**
    *   Yes: Short interval (acute) or appropriate for condition.
    *   No: Long delay allowing disease progression/regression.
2.  **Did all patients receive the same reference standard?**
    *   Yes: Consistent reference standard.
    *   No: Different standards used (verification bias).
3.  **Were all patients included in the analysis?**
    *   Yes: All enrolled patients accounted for.
    *   No: Withdrawals or missing data not explained.
