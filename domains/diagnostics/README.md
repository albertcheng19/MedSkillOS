# Diagnostics Domain

The diagnostics domain provides structured clinical reasoning skills based on diagnostic workflow concepts including chief complaint, present illness history, red flags, differential diagnosis, evidence mapping, uncertainty, and role-specific communication.

This domain is knowledge-light and standards-heavy.

It does not copy diagnostic textbooks.

It does not provide autonomous diagnosis.

It is intended for medical education, clinician-reviewed workflow support, safety-aware reasoning, and expert-reviewed agent development.
