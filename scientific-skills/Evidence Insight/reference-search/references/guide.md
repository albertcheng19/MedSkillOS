# Literature Search Checklist

## Input Checklist

- Research topic and core questions
- Keywords and synonym list
- Database preferences (PubMed/Scopus/Web of Science, etc.)
- Time range, language, and document type

## Output Checklist

- Final search strings and database list
- De-duplicated literature list (Title/Author/Journal/Year/DOI/Link/Abstract)
- Search statistics overview (Hits, de-duplicated count)

## Output Format

- Output the literature list in table format.
- List search strings and search dates separately.

## Quality Check

- Search strings cover synonyms and abbreviations.
- Record the search date for each database.
- De-duplication rules are clear and reproducible.

## Common Issues

- Omissions caused by using only a single database.
- Failure to record search strings and dates.
