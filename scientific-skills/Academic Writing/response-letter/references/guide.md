# Reviewer Response Assistant Reference Checklist

## Input Checklist

- List of reviewer comments
- Revised manuscript or list of changes
- Manuscript version information

## Output Checklist

- Word response document (.docx)
- List of changes/locations

## Output Format

- An introductory summary paragraph addressed to the editor at the beginning of the document, summarizing the main revisions.
- The main body formatted as a point-by-point response:
  - Reviewer's Comment
  - Response
  - Changes in text
- Indicate page/paragraph/line numbers in "Changes in text".
- If the main text has been modified, excerpt the revised paragraphs using indentation or blockquote formatting.

## Quality Check

- Every comment has a corresponding response.
- Locations of changes are clearly marked.

## Common Issues

- Missing comments or disorganized numbering.
- Response tone is not professional enough.
