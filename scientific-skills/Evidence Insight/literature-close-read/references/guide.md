# Literature Intensive Reading Reference Checklist

## Input Checklist

- Full text of the literature (Markdown, usually containing `## Page XX` pagination)
- Research questions or focus areas (Optional)

## Output Checklist

- Structured abstract (Problem/Method/Results/Conclusion)
- Interpretation of key data and charts
- Key points for reproduction and list of limitations

## Output Format

- Use sections or tables to output structured content.
- Annotate key data with original numerical values.
- Output as a UTF-8 encoded Markdown document.

## Quality Check

- Methodological and statistical details are fully recorded.
- Results and conclusions correspond to evidence.

## Common Issues

- Summarizing only conclusions without data support.
- Ignoring methodological details leading to irreproducibility.
