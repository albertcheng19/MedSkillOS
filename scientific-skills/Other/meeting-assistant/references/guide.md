# Meeting Assistant Reference Checklist

## Input Checklist

- Meeting minutes/Transcription text
- Agenda or timeline information

## Output Checklist

- Timeline summary
- Decision list
- Action items list (Owner/Deadline)

## Output Format

- Output in chronological sections.
- Present action items in a table.

## Quality Check

- Action items include owners and deadlines.
- Clear distinction between decisions and items to be confirmed.

## Common Issues

- Missing key decisions or action items.
- Confused timeline.
