# CSL Style Acquisition and Usage

## Common Journal Styles

- NEJM: `new-england-journal-of-medicine.csl`
- Lancet: `the-lancet.csl`
- Nature: `nature.csl`

## How to Download

1. Visit the Zotero Style Repository: `https://www.zotero.org/styles`
2. Search for the journal name and download the `.csl` file.
3. Save the `.csl` file to your project directory (e.g., `styles/`).
4. Use `--style` to specify the style path in your script.

## Custom Styles

- To edit a style, copy the original `.csl` file before making modifications.
- Keep the filename after modification to track the source (you can add a custom prefix to the filename).
