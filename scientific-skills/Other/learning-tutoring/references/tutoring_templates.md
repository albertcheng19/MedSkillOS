# Learning Tutoring Output Templates

The following templates are used to generate learning plans, practice questions, and answer explanations. Adapt them according to the subject and the learner's level, but maintain a clear structure.

## Learning Plan Template

Suggested Output Structure:

1. Learner Profile
2. Learning Objectives
3. Weekly Plan (by week or stage)
4. Milestones and Acceptance Criteria
5. Review and Adjustment Suggestions

Example Structure:

- Learner Profile: {Subject} / {Level} / {Study Duration} per week
- Learning Objectives: {Objective 1}, {Objective 2}
- Weekly Plan:
  - Week 1: {Focus}; Tasks: {Reading} / {Practice} / {Review}
  - Week 2: ...
- Acceptance Criteria: Able to explain {Concept}, complete {Task}
- Adjustment Strategy: Fine-tune the pace based on incorrect answers and feedback

## Practice Question Template

Suggested Output Structure:

- Question Number
- Question Type (choice / short_answer / application)
- Question
- Options (if multiple choice)
- Answer
- Explanation
- Difficulty and Knowledge Point Tags

Example Structure:

1) Type: choice
Question: {Stem}
Options: A. ... B. ... C. ... D. ...
Answer: A
Explanation: {Why A is correct, why other options are inappropriate}
Difficulty: Basic
Tags: {topic}

## Answer Explanation Template

Explanations should include:

- Correct Logic: Provide key reasoning or steps
- Common Pitfalls: Point out common misunderstandings
- Transfer Tips: How to apply the knowledge to other scenarios

Example Structure:

Explanation:
1. Correct Logic: ...
2. Common Pitfalls: ...
3. Transfer Tips: ...
