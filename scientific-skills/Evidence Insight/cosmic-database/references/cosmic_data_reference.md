# COSMIC Data Reference

## Included Data Types

The skill supports access to the following COSMIC data categories:

### 1. Cancer Gene Census (CGC)
Curated list of genes with substantial evidence of involvement in cancer.

### 2. Mutational Signatures
- **SBS**: Single Base Substitutions
- **DBS**: Double Base Substitutions
- **ID**: Small Insertions and Deletions

### 3. Genomic Data
- Copy number variants (CNV)
- Gene fusions
- Gene expression data
- Methylation data

## Versioning
- **GRCh37**: Legacy genome build.
- **GRCh38**: Current genome build (Recommended).
