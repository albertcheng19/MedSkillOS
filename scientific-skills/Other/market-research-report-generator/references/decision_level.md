# Decision Level Framework

Use this framework to tailor the report's tone and depth.

## 1. C-Level (CEO, CFO, CSO)
- **Tone**: Strategic, concise, bottom-line focused.
- **Format**: Executive Summary is critical. Use bullet points and high-level charts.
- **Focus**: Financial impact, strategic risks, long-term vision.
- **Avoid**: Excessive technical jargon, operational minutiae.

## 2. VP / Director Level
- **Tone**: Tactical, balanced, actionable.
- **Format**: Structured sections with clear headings.
- **Focus**: Implementation, resource allocation, competitive positioning, quarterly/yearly goals.

## 3. Manager / Operational Level
- **Tone**: Practical, detailed, instructional.
- **Format**: Step-by-step guides, detailed comparison tables.
- **Focus**: Execution, process optimization, vendor selection.

## 4. R&D / Technical Specialist
- **Tone**: Technical, precise, data-heavy.
- **Format**: Whitepaper style, citations, technical specs.
- **Focus**: Feasibility, specifications, clinical data, architectural details.
