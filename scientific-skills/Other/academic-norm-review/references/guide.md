# Academic Standards Reference Checklist

## Input Checklist

- Full paper
- Reference list
- Journal or school formatting requirements

## Output Checklist

- List of compliance issues
- Paraphrasing and citation suggestions
- Revised paragraphs or text
- Formatting consistency notes

## Output Format

- List of issues organized by location.
- Suggestions should include the direction for revision.
- Provide revised text in the same language as the original.
- The language of suggestions should remain consistent with the input language.

## Quality Check

- Citation format is consistent with journal standards.
- Abbreviations are defined upon first appearance.

## Common Issues

- Ignoring synonymous expressions leading to repetition.
- Inconsistency between in-text citations and the reference list.
