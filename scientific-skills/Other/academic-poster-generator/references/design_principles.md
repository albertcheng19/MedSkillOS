## Typography

### Font Selection

**Recommended Fonts:**
- Arial / Helvetica (clean, readable)
- Calibri (modern, professional)
- Trebuchet MS (friendly, approachable)
- Open Sans / Lato (web-safe alternatives)

**Avoid:**
- Times New Roman (too traditional, hard to read at distance)
- Comic Sans (unprofessional)
- Decorative fonts (hard to read)
- Fonts with serifs (reduced readability)

### Font Sizes

**Standard Poster Typography:**
```
Title:          72-120pt  (visible from 10-15 feet)
Authors:         36-48pt   (visible from 6-8 feet)
Section Headers: 48-72pt   (visible from 6-8 feet)
Subheaders:      32-40pt   (visible from 4-6 feet)
Body Text:       24-36pt   (readable from 2-4 feet)
Captions:        18-24pt   (readable from 1-2 feet)
```

**Testing Font Sizes:**
1. Print at 25% scale (A0 → A4)
2. View from 2-3 feet
3. If readable, appropriate for full-size poster

### Font Usage Best Practices

✓ **Do:**
- Use consistent font throughout
- Limit to 2-3 font families maximum
- Use bold for emphasis sparingly
- Use uppercase only for titles
- Ensure high contrast with background

✗ **Don't:**
- Mix too many font styles
- Use italic for body text (hard to read)
- Use underlining (looks like web links)
- Use font size smaller than 24pt for body text
- Use all-caps for body text

## Color and Contrast

### Color Palettes

**Academic Professional:**
```
Primary:   #003366 (Navy Blue)
Secondary: #336699 (Medium Blue)
Accent:     #990033 (Deep Red)
Background: #FFFFFF (White)
Text:      #000000 (Black)
```

**Modern Clean:**
```
Primary:   #2E4053 (Dark Slate)
Secondary: #1A936F (Emerald Green)
Accent:     #F6AE2D (Warm Orange)
Background: #FFFFFF (White)
Text:      #2E4053 (Dark Slate)
```

**Color-Blind Friendly (Viridis-style):**
```
Primary:   #440154 (Deep Purple)
Secondary: #21918C (Teal)
Tertiary:  #FDE725 (Yellow)
Background: #FFFFFF (White)
Text:      #000000 (Black)
```

### Contrast Requirements

**WCAG AA Standard:**
- Normal text: 4.5:1 contrast ratio
- Large text (18pt+): 3:1 contrast ratio
- Graphical elements: 3:1 contrast ratio

**WCAG AAA Standard:**
- Normal text: 7:1 contrast ratio
- Large text (18pt+): 4.5:1 contrast ratio

### Color Testing Tools

- WebAIM Contrast Checker: https://webaim.org/resources/contrastchecker/
- ColorBrewer: http://colorbrewer2.org/
- Coblis Color Blindness Simulator: https://www.color-blindness.com/coblis-color-blindness-simulator/

### Color Usage Guidelines

✓ **Do:**
- Use institutional colors for branding
- Test with color blindness simulator
- Maintain consistent color scheme
- Use white space effectively
- Consider printing (CMYK conversion)

✗ **Don't:**
- Use red-green combinations (most problematic)
- Use neon or fluorescent colors
- Overuse bright colors (eye strain)
- Use low contrast combinations
- Forget to test on different screens

## Layout and Visual Hierarchy

### Column Layouts

**2-Column Layout:**
- Best for: Simple research, focused topics
- Column width: 40-45% of page
- Gap between columns: 8-10% of page

**3-Column Layout (Most Common):**
- Best for: Standard research posters
- Column width: 28-30% of page
- Gap between columns: 5-7% of page

**4-Column Layout:**
- Best for: Complex, information-dense posters
- Column width: 20-22% of page
- Gap between columns: 4-5% of page

### Visual Flow

**Z-Pattern Reading:**
1. Top-left: Title and introduction
2. Top-right: Key findings
3. Middle-left: Methods
4. Middle-right: Results
5. Bottom: Conclusions and references

**F-Pattern Reading:**
1. Top: Title and authors
2. Left column: Introduction → Methods
3. Right column: Results → Conclusions
4. Bottom: References and acknowledgments

### White Space

**Effective White Space:**
- Use 15-20% of poster for white space
- Consistent spacing between blocks (15-20mm)
- 10-15mm margin around content
- White space improves readability and visual appeal

### Visual Hierarchy

**Hierarchy Levels:**
```
Level 1: Title (largest, boldest)
Level 2: Section headers
Level 3: Subheaders
Level 4: Body text
Level 5: Captions and footnotes
```

**Emphasis Techniques:**
- Size: Larger for more important elements
- Color: Accent color for key points
- Position: Eye-catching locations (top-left, center)
- White space: More space around important elements

## Visual Elements

### Figures and Images

**Image Quality:**
- Minimum 300 DPI at final print size
- Vector graphics (PDF/SVG) preferred
- Use JPEG for photographs, PNG for graphics
- Test at 100% zoom before printing

**Figure Placement:**
- Place figures near related text
- Use consistent sizing
- Add clear, concise captions
- Reference figures in text

**Figure Best Practices:**
- 3-6 main figures maximum
- 40-50% of poster should be visual
- Use high-contrast labels
- Remove unnecessary borders
- Maintain consistent style

### Tables

**Table Design:**
- Simple is better: limit to 3-5 columns
- Use clean, readable fonts
- Add borders for readability
- Highlight key findings
- Keep concise (3-5 rows)

**Table Tips:**
- Consider converting to figure for clarity
- Use bold for headers
- Add brief caption
- Reference in text
- Ensure high contrast

### QR Codes

**QR Code Placement:**
- Bottom-right corner (standard location)
- Minimum 2×2cm size
- High contrast (black on white)
- Test with multiple scanners

**QR Code Content:**
- Link to full paper
- Link to supplementary materials
- Link to GitHub repository
- Link to video presentation
- Add "Scan for more info" label

## Content Guidelines

### Text Length

**Recommended Word Counts:**
```
Introduction:  100-150 words
Methods:       100-150 words
Results:       200-300 words
Discussion:     150-200 words
Conclusions:    50-100 words
Total:         600-900 words
```

### Writing Style

**Poster Writing:**
- Use active voice
- Present tense for established facts
- Past tense for methods and results
- Clear, concise language
- Avoid jargon when possible
- Define acronyms first use

### Bullet Points

**Effective Bullets:**
- Use bullet points, not paragraphs
- Limit to 5-7 bullets per section
- Keep bullets concise (10-20 words)
- Parallel structure
- Remove unnecessary words

### Section Structure

**Essential Sections:**
1. Title, Authors, Affiliations
2. Introduction/Background
3. Methods/Approach
4. Results (with figures)
5. Discussion/Interpretation
6. Conclusions
7. References (abbreviated)
8. Acknowledgments

**Optional Sections:**
- Future Work
- Limitations
- Contact Information
- QR Code

## Accessibility

### Visual Impairment

**Design for Low Vision:**
- Large, clear fonts (minimum 24pt body)
- High contrast (4.5:1 minimum)
- Avoid complex backgrounds
- Clear visual hierarchy
- Consistent layout

### Color Blindness

**Color-Blind Friendly Design:**
- Avoid red-green combinations
- Use patterns or shapes in addition to color
- Test with color blindness simulator
- Use texture for emphasis, not just color
- Label colors in legends

### International Audiences

**Language Considerations:**
- Simple, clear language
- Define technical terms
- Avoid idioms and colloquialisms
- Consider multilingual QR codes

### Assistive Technology

**Digital Accessibility:**
- Alt text for images
- Logical reading order
- Keyboard navigation
- Screen reader compatible
- Tagged PDF structure

## Common Mistakes

### Design Mistakes

❌ **Typography:**
- Font size too small (<24pt body text)
- Too many different fonts
- Hard-to-read fonts (serif, decorative)
- Inconsistent font usage

❌ **Color:**
- Low contrast text/background
- Too many colors (rainbow effect)
- Red-green combinations
- No color planning

❌ **Layout:**
- No visual hierarchy
- Inconsistent spacing
- Too much text (wordy poster)
- Cluttered appearance

### Content Mistakes

❌ **Text:**
- Copy-pasting full abstract
- Using paragraphs instead of bullets
- Too much jargon
- Unclear organization

❌ **Visual Elements:**
- Low-resolution images
- Too many figures
- Inconsistent figure styles
- Missing captions

### Technical Mistakes

❌ **Formatting:**
- Wrong poster size
- Text cut off at edges
- Inconsistent alignment
- Missing sections

❌ **Printing:**
- Wrong color space (RGB vs CMYK)
- Fonts not embedded
- File too large
- No test print

## Quick Checklist

**Before Finalizing Poster:**

- [ ] Font sizes meet minimum requirements
- [ ] High contrast colors throughout
- [ ] Consistent spacing and alignment
- [ ] 3-6 high-quality figures included
- [ ] Text organized as bullet points
- [ ] Word count 600-900 total
- [ ] All sections present
- [ ] QR code works (if used)
- [ ] Test print at 25% scale
- [ ] Viewable from 2-4 feet
- [ ] No spelling/grammar errors
- [ ] Authors and affiliations correct
- [ ] Contact information visible
