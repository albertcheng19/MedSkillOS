# ROB2 Assessment Guidelines

## Domain 1: Randomization Process
**1.1 Was the allocation sequence random?**
- Answer 'Yes' if a random component was used (computer-generated, dice, etc.).
- Answer 'No' if systematic (dates, alternation).
- Answer 'No information' if not stated.

**1.2 Was the allocation sequence concealed?**
- Answer 'Yes' if central allocation or opaque sealed envelopes.
- Answer 'No' if open list or transparent envelopes.

**1.3 Did baseline differences suggest a problem?**
- Answer 'Yes' if substantial imbalances.

**Risk-of-bias judgement (Domain 1):**
- Low risk: 1.2 is Y/PY AND 1.1 is Y/PY/NI AND 1.3 is N/PN/NI.
- Some concerns: 1.2 is Y/PY AND 1.1 is N/PN.
- High risk: 1.2 is N/PN OR (1.2 is NI AND 1.3 is Y/PY).

## Domain 2: Deviations from Intended Interventions
**2.1 Were participants aware of their assigned intervention?**
**2.2 Were carers/people delivering interventions aware?**
**2.3 If aware: Were there deviations arose because of the trial context?**
**2.4 If deviations: Were they likely to have affected the outcome?**
**2.5 If deviations: Were they balanced between groups?**
**2.6 Was an appropriate analysis used (ITT)?**
**2.7 If inappropriate analysis: Potential for substantial impact?**

**Risk-of-bias judgement (Domain 2):**
- Low risk: 2.1/2.2 are N/PN OR (2.1/2.2 are Y/PY/NI AND 2.3 is NA/Y/PY AND 2.6 is Y/PY).
- Some concerns: 2.6 is N/PN/NI.
- High risk: Substantial impact of failure to analyze correctly.
