# Experimental Protocol Standardization Reference Checklist

## Input Checklist

- Original experimental steps
- Material/Reagent/Equipment information
- Key parameters and precautions

## Output Checklist

- Standardized protocol document
- Checklist of key control points and precautions

## Output Format

- Clear step numbering and unified units.
- Includes material list and safety information.

## Quality Check

- Step sequence is executable and unambiguous.
- Key parameters are complete.

## Common Issues

- Missing key parameters leading to irreproducibility.
- Disordered step sequence.
