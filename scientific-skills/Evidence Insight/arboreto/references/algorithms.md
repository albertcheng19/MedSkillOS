# Algorithms

## GRNBoost2
- **Method**: Gradient Boosting Machine (GBM).
- **Pros**: Faster and more scalable than GENIE3. Recommended for large datasets (e.g., single-cell RNA-seq).
- **Cons**: Stochastic nature might yield slightly different results across runs (set seed for reproducibility).

## GENIE3
- **Method**: Random Forest.
- **Pros**: Classic, well-cited algorithm.
- **Cons**: Slower on large datasets.
