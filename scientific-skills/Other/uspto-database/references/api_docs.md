# API References

## PatentSearch API
- **Endpoint**: `https://api.patentsview.org/patents/query`
- **Method**: POST
- **Query Syntax**: ElasticSearch-style (`_and`, `_or`, `_text_all`).

## PEDS API (Patent Examination Data System)
- **Endpoint**: `https://ped.uspto.gov/api/queries`
- **Data**: JSON/XML transaction history.
- **Codes**: `CTNF` (Non-Final Rejection), `NOA` (Notice of Allowance).

## TSDR API (Trademark Status)
- **Endpoint**: `https://tsdr.uspto.gov/`
- **Function**: Check trademark status and documents.
