# Electrooculography (EOG) Analysis

## Overview

Electrooculography (EOG) measures eye movements and blinks by detecting the potential difference generated by changes in eye position. EOG is commonly used in sleep studies, attention research, reading analysis, and artifact correction for EEG.

## Main Processing Pipeline

### eog_process()

Automated EOG signal processing pipeline.

```python
signals, info = nk.eog_process(eog_signal, sampling_rate=500, method='neurokit')
```

**Processing Steps:**
1. Signal cleaning (filtering)
2. Blink detection
3. Blink rate calculation

**Return Value:**
- `signals`: A DataFrame containing the following:
  - `EOG_Clean`: Filtered EOG signal
  - `EOG_Blinks`: Binary blink markers (0/1)
  - `EOG_Rate`: Instantaneous blink rate (blinks/min)
- `info`: A dictionary containing blink indices and parameters

**Methods:**
- `'neurokit'`: NeuroKit2 optimized method (default)
- `'agarwal2019'`: Algorithm by Agarwal et al. (2019)
- `'mne'`: MNE-Python method
- `'brainstorm'`: Brainstorm toolbox method
- `'kong1998'`: Method by Kong et al. (1998)

## Preprocessing Functions

### eog_clean()

Prepare raw EOG signals for blink detection.

```python
cleaned_eog = nk.eog_clean(eog_signal, sampling_rate=500, method='neurokit')
```

**Methods:**
- `'neurokit'`: Butterworth filter optimized for EOG
- `'agarwal2019'`: Alternative filtering approach
- `'mne'`: MNE-Python preprocessing
- `'brainstorm'`: Brainstorm method
- `'kong1998'`: Kong's method

**Typical Filter Settings:**
- Low-pass: 10-20 Hz (removes high-frequency noise)
- High-pass: 0.1-1 Hz (removes DC drift)
- Preserve blink waveforms (typical duration 100-400 ms)

**EOG Signal Characteristics:**
- **Blinks**: Large amplitude, typical waveform (200-400 ms)
- **Saccades**: Rapid step-like deflections (20-80 ms)
- **Smooth pursuit**: Slow ramp-like changes
- **Baseline**: Remains stable during eye fixation

## Blink Detection

### eog_peaks()

Detect eyelid blinks in EOG signals.

```python
blinks, info = nk.eog_peaks(cleaned_eog, sampling_rate=500, method='neurokit',
                            threshold=0.33)
```

**Methods:**
- `'neurokit'`: Based on amplitude and duration criteria (default)
- `'mne'`: MNE-Python blink detection
- `'brainstorm'`: Brainstorm method
- `'blinker'`: BLINKER algorithm (Kleifges et al., 2017)

**Key Parameters:**
- `threshold`: Amplitude threshold (proportion of maximum amplitude)
  - Typical values: 0.2-0.5
  - Lower values: More sensitive (may include false positives)
  - Higher values: More conservative (may miss small blinks)

**Return Value:**
- A dictionary with the `'EOG_Blinks'` key, containing blink peak indices

**Blink Characteristics:**
- **Frequency**: 15-20 times/minute (resting, comfortable state)
- **Duration**: 100-400 ms (average ~200 ms)
- **Amplitude**: Varies with electrode placement and individual differences
- **Waveform**: Biphasic or triphasic

### eog_findpeaks()

Low-level blink detection using various algorithms.

```python
blinks_dict = nk.eog_findpeaks(cleaned_eog, sampling_rate=500, method='neurokit')
```

**Use Cases:**
- Custom parameter fine-tuning
- Algorithm comparison
- Research method development

## Feature Extraction

### eog_features()

Extract features of individual blinks.

```python
features = nk.eog_features(signals, sampling_rate=500)
```

**Calculated Features:**
- **Amplitude Velocity Ratio (AVR)**: Peak velocity / Amplitude
  - Used to distinguish blinks from artifacts
- **Blink Amplitude Ratio**: Consistency of blink amplitudes
- **Duration Metrics**: Blink duration statistics (mean, SD)
- **Peak Amplitude**: Maximum deflection
- **Peak Velocity**: Maximum rate of change

**Use Cases:**
- Blink quality assessment
- Drowsiness detection (blink duration increases when sleepy)
- Neurological assessment (blink dynamics change in disease states)

### eog_rate()

Calculate blink frequency (blinks per minute).

```python
blink_rate = nk.eog_rate(blinks, sampling_rate=500, desired_length=None)
```

**Methods:**
- Calculate instantaneous blink intervals
- Convert to blinks per minute
- Interpolate to match signal length

**Typical Blink Rates:**
- **Resting**: 15-20 blinks/min
- **Reading/Visual tasks**: 5-10 blinks/min (suppressed)
- **Conversation**: 20-30 blinks/min
- **Stress/Dry eyes**: >30 blinks/min
- **Drowsiness**: Fluctuating, with longer blink durations

## Analysis Functions

### eog_analyze()

Automatically select event-related or interval-related analysis.

```python
analysis = nk.eog_analyze(signals, sampling_rate=500)
```

**Mode Selection:**
- Duration < 10 seconds → event-related
- Duration ≥ 10 seconds → interval-related

### eog_eventrelated()

Analyze blink patterns relative to specific events.

```python
results = nk.eog_eventrelated(epochs)
```

**Calculated Metrics (per epoch):**
- `EOG_Blinks_N`: Number of blinks during the epoch
- `EOG_Rate_Mean`: Average blink rate
- `EOG_Blink_Presence`: Binary (whether a blink occurred)
- Temporal distribution of blinks throughout the epoch

**Use Cases:**
- Assess blink-locked ERP contamination
- Attention and engagement during stimuli
- Visual task difficulty (blinks are suppressed during high-difficulty tasks)
- Spontaneous blinks after stimulus offset

### eog_intervalrelated()

Analyze blink patterns over long-term intervals.

```python
results = nk.eog_intervalrelated(signals, sampling_rate=500)
```

**Calculated Metrics:**
- `EOG_Blinks_N`: Total number of blinks
- `EOG_Rate_Mean`: Average blink rate (blinks/min)
- `EOG_Rate_SD`: Blink rate variability
- `EOG_Duration_Mean`: Average blink duration (if available)
- `EOG_Amplitude_Mean`: Average blink amplitude (if available)

**Use Cases:**
- Resting-state blink patterns
- Drowsiness or fatigue monitoring (increased duration)
- Sustained attention tasks (decreased blink rate)
- Dry eye assessment (increased blink rate, incomplete blinks)

## Practical Application Considerations

### Sampling Rate Recommendations
- **Minimum**: 100 Hz (basic blink detection)
- **Standard**: 250-500 Hz (research applications)
- **High Resolution**: 1000 Hz (detailed waveform analysis, saccade studies)
- **Sleep Studies**: Typically 200-250 Hz

### Recording Duration
- **Blink Detection**: Any duration (≥1 blink)
- **Blink Rate Estimation**: ≥60 seconds for stable assessment
- **Event-related**: Depends on paradigm (seconds per trial)
- **Sleep EOG**: Several hours (overnight)

### Electrode Placement

**Standard Configurations:**

**Horizontal EOG (HEOG):**
- Two electrodes: Outer canthi (outer corners) of the left and right eyes
- Measures horizontal eye movements (saccades, smooth pursuit)
- Bipolar recording (Left - Right)

**Vertical EOG (VEOG):**
- Two electrodes: Above and below a single eye (usually the right eye)
- Measures vertical eye movements and blinks
- Bipolar recording (Upper - Lower)

**Sleep EOG:**
- Typically uses slightly different placement (temporal regions)
- E1: 1 cm lateral and 1 cm inferior to the outer canthus of the left eye
- E2: 1 cm lateral and 1 cm superior to the outer canthus of the right eye
- Captures both horizontal and vertical movements simultaneously

**EEG Contamination Removal:**
- Frontal electrodes (Fp1, Fp2) can serve as alternative channels for EOG
- ICA-based EOG artifact removal is commonly used in EEG preprocessing

### Common Issues and Solutions

**Electrode Issues:**
- Poor contact: Low amplitude, high noise
- Skin preparation: Cleaning, light abrasion
- Conductive gel: Ensure good contact

**Artifacts:**
- Muscle activity (especially frontalis): High-frequency noise
- Movement: Cable artifacts, head movements
- Electrical noise: 50/60 Hz power line interference (requires proper grounding)

**Saturation:**
- Large-scale saccades may saturate the amplifier
- Adjust gain or voltage range
- More common in low-resolution systems

### Best Practices

**Standard Workflow:**
```python
# 1. Clean signal
cleaned = nk.eog_clean(eog_raw, sampling_rate=500, method='neurokit')

# 2. Detect blinks
blinks, info = nk.eog_peaks(cleaned, sampling_rate=500, method='neurokit')

# 3. Extract features
features = nk.eog_features(signals, sampling_rate=500)

# 4. Comprehensive processing (alternative)
signals, info = nk.eog_process(eog_raw, sampling_rate=500)

# 5. Analysis
analysis = nk.eog_analyze(signals, sampling_rate=500)
```

**EEG Artifact Correction Workflow:**
```python
# Option 1: Regression-based removal
# Identify EOG components from cleaned EOG signals
# Regress out EOG components from EEG channels

# Option 2: ICA-based removal (recommended)
# 1. Run ICA on EEG data including EOG channels
# 2. Identify EOG-related ICA components
# 3. Remove EOG components from EEG data
# NeuroKit2 integrates with MNE to support this workflow
```

## Clinical and Research Applications

**EEG Artifact Correction:**
- Blinks contaminate frontal EEG channels
- ICA or regression methods can remove EOG artifacts
- Crucial for ERP studies

**Sleep Staging:**
- Rapid Eye Movements (REMs) during REM sleep
- Slow rolling eye movements during drowsiness
- Sleep onset and stage transitions

**Attention and Cognitive Load:**
- Blink rate is suppressed during high-demand tasks
- Blinks often cluster at task boundaries (natural breakpoints)
- Spontaneous blinks as indicators of attention shifts

**Fatigue and Drowsiness Monitoring:**
- Blink duration increases when drowsy
- Slower eyelid closure
- Partial or incomplete blinks
- Driver monitoring applications

**Reading and Visual Processing:**
- Blinks are suppressed during reading
- Eye movements during saccades (line changes)
- Impact of fatigue on reading efficiency

**Neurological Disorders:**
- **Parkinson's Disease**: Reduced spontaneous blink rate
- **Schizophrenia**: Increased blink rate
- **Tourette Syndrome**: Excessive blinking (tics)
- **Dry Eye Syndrome**: Increased blink rate, incomplete blinks

**Emotion and Social Cognition:**
- Blink synchrony in social interactions
- Emotional regulation of blink rate
- Blink-related potentials in ERPs

**Human-Computer Interaction:**
- Eye-tracking preprocessing
- Attention monitoring
- User engagement assessment

## Types of Eye Movements Detectable by EOG

**Blinks:**
- Large amplitude, short duration (100-400 ms)
- Primary focus of NeuroKit2
- Vertical EOG is most sensitive

**Saccades:**
- Rapid, ballistic eye movements (20-80 ms)
- Step-like voltage deflections
- Horizontal or vertical
- Requires higher sampling rates for detailed analysis

**Smooth pursuit:**
- Slow tracking of moving objects
- Ramp-like voltage changes
- Lower amplitude than saccades

**Fixations:**
- Stable gaze
- Baseline EOG with micro-oscillations
- Variable duration (typically 200-600 ms in reading)

**Note:** Detailed saccade/fixation analysis usually requires eye trackers (infrared, video-based). EOG is suitable for blinks and coarse eye movement analysis.

## Interpretation Guidelines

**Blink Rate:**
- **Normal resting**: 15-20 blinks/min
- **<10 blinks/min**: Visual task engagement, focus
- **>30 blinks/min**: Stress, dry eyes, fatigue
- **Context-dependent**: Depends on task demands, lighting, screen use

**Blink Duration:**
- **Normal**: 100-400 ms (average ~200 ms)
- **Prolonged**: Drowsiness, fatigue (>500 ms)
- **Short**: Normal alertness

**Blink Amplitude:**
- Varies with electrode position and individual
- Within-subject comparisons are most reliable
- Incomplete blinks: Reduced amplitude (dry eyes, fatigue)

**Temporal Patterns:**
- **Blink bursts**: Transitions between tasks or cognitive states
- **Inhibitory blinking**: Active visual processing, sustained attention
- **Post-stimulus blinking**: After completing visual processing
