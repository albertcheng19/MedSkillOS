# Academic Standards Checklist

## Citations and References
- [ ] In-text citations match the reference list
- [ ] Citation format follows journal requirements

## Abbreviations and Terminology
- [ ] Abbreviations are defined upon first mention
- [ ] Terminology is used consistently

## Plagiarism and Paraphrasing
- [ ] High-similarity passages have been addressed
- [ ] Direct quotes are properly attributed

## Output Requirements
- [ ] Suggested language remains consistent with the original input language
- [ ] Provide revised paragraphs or text
